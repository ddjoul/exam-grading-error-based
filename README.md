# Критериальная система оценивания экзаменационных работ на основе ошибки относительно эталона

## Описание

Проект предлагает объективную систему оценивания задач с числовым результатом (например, молярная масса вещества) на основе ошибки относительно эталонного значения, а не распределения результатов группы.

## Данные

Реальные экзаменационные результаты за 3 года:
* значения, рассчитанные студентами;
* оценки преподавателей;
* данные о повторных попытках.

Эталонное значение определяется средним результатом группы преподавателей.

## Этапы анализа
1. Проверка распределения результатов.
2. Анализ применения 3σ и 1.5 IQR.
3. Расчет абсолютной и относительной ошибки.
4. Формирование шкалы оценок (0–5) по относительной ошибке.
5. Сравнение с исходной системой оценивания.

## Результаты
* Статистические критерии (3σ, IQR) зависят от состава группы и непостоянны.
* Фактические оценки связаны с ошибкой, но границы оценок не формализованы.
* Предложена шкала по относительной ошибке, обеспечивающая объективность и воспроизводимость.
* Расхождения с исходной системой связаны с неоднородностью критериев, а не качеством работы студентов.

## Вывод
Для задач с известным эталоном оценивание должно базироваться на ошибке, обеспечивая интерпретируемость и повторяемость.
